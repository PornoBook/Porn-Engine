# HTML engine for an adult website with a movie database

**Site Configuration**

1. Open the config folder.
2. Configure the files there:
   - **siteinfo.js** - description of the site
   - **ads.js** - advertising banner number 1
   - **ads2.js** - advertising banner number 2
   - **stats.js** - statistics for the site; instructions for connection can be found in the JS script
   - **video-skins.js** - player style configuration
   - **pages.js** - pages button
